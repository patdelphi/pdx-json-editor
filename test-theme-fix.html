Monaco Editor 主题切换修复测试

测试步骤：

  1. 启动开发服务器：npm run dev
  2. 在浏览器中打开应用
  3. 观察当前Monaco Editor的主题（默认应该是明亮主题）
  4. 点击右上角的主题切换按钮（月亮图标）
  5. 观察Monaco Editor是否立即切换到暗黑主题
  6. 再次点击主题切换按钮（太阳图标）
  7. 观察Monaco Editor是否立即切换回明亮主题

预期结果：

  • ✅ Monaco Editor主题应该与应用主题同步切换
  • ✅ 切换应该是即时的，无需重新加载编辑器
  • ✅ 编辑器内容和光标位置应该保持不变
  • ✅ 不应该有任何控制台错误

修复内容：

1. 移除了编辑器重新创建逻辑
- 不再使用 editorKey 强制重新渲染
- 改为直接调用 Monaco API 更新主题

2. 优化了Monaco主题管理器
- 添加了异步主题设置，确保Monaco完全初始化
- 添加了isReady()检查方法

3. 移除了theme prop冲突
- Editor组件不再传递theme prop
- 完全依赖Monaco主题管理器统一管理

如果仍有问题，请检查：

  • 浏览器开发者工具的控制台是否有错误信息
  • Monaco Editor是否完全加载完成
  • 主题切换按钮是否正常工作（应用主题是否切换）
